# Delivery Exception Management Dashboard

## Overview

The Delivery Exception Management Dashboard is a web-based application developed for GreyAtom Logistics Pvt. Ltd. It helps operations managers record, monitor, and resolve delivery-related issues in an organized manner. The project is built using HTML, CSS, and JavaScript without any frameworks or external libraries.

## Features

* Add new delivery exception reports
* View all reported issues in a dashboard table
* Filter issues by issue type and status
* Mark issues as resolved
* Delete issues with confirmation
* Display counts of open and resolved issues
* Highlight high-priority issues for quick attention

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## How to Run

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using Live Server or any modern web browser.
